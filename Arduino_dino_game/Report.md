## AIM
To design and implement a Dinosaur Runner Game using an Arduino board and a 16×2 I2C LCD display, where the player controls a dinosaur to jump over randomly generated obstacles using a push button while tracking the score in real time.

## COMPONENTS

- Arduino Uno / Arduino Nano
- 16×2 I2C LCD Display
- Push Button
- Buzzer
- Breadboard
- Jumper Wires
- USB Cable 

## CIRCUIT CONNECTIONS

 VCC - 5V
 GND - GND
 SDA - A4
 SCL - A5
Push Button - D5
Push Button (Other Terminal)-  GND
Buzzer Positive (+) - D7
Buzzer Negative (−) -GND
