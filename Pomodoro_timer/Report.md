# POMODORO TIMER REPORT

## 1. AIM
 It is designed as a simple productivity tool while demonstrating Arduino programming and LCD interfacing

 ## 2. COMPONENTS USED
  - Arduino UNO
  - 16 * 2 LCD with I2C Module
  - 4 Jumper wires
  - USB Cable

## 3. CIRCUIT CONNECTIONS
 - VCC - 5V
 - GND - GND
 - SDA - A4
 - SCL - A5
